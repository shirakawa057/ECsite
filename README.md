# ショッピングサイト
 
# 概要
 
データベースの操作に関する基本機能を備えたアプリケーション。
作業内容について登録、更新、削除、閲覧できる。
基本的なデータベースを操作するCURD機能に加え、ログインユーザーの情報を修正するなどのセキュリティに関する基本機能を実装した。
